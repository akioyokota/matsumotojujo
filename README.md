# 10 Stories Stay Matsumoto ドキュメント

10 Stories Stay Matsumoto の運用に関するマニュアルとレポートをまとめたリポジトリです。清掃やお客様対応などの現場業務の「型」を共有し、誰が担当しても一定の品質を保てるようにすることを目的としています。

## ディレクトリ構成

```
.
├── マニュアル/   現場業務の手順書
└── レポート/     業務改善に関するレポート
```

## 使い方

- 各マニュアルは Markdown 形式です。GitHub 上でそのまま閲覧できます。
- 手順を更新する場合は、該当する Markdown ファイルを直接編集してください。
- マニュアル内の `【画像：〇〇】` `【画像必要：〇〇】` は、掲載予定・掲載が必要な画像の目印です。

## 更新の進め方

1. 新しいブランチを作成する
2. 対象の Markdown ファイルを編集する
3. 変更内容がわかるコミットメッセージでコミットする
4. プルリクエストを作成してレビューを受ける
