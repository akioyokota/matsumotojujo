# 10 Stories Stay Matsumoto お客様対応ルール

## このマニュアルの目的

客室への入室・館内ですれ違った際など、**Stay でお客様と直接顔を合わせる場面**での対応を統一し、誰が担当しても同じ品質のおもてなしを提供するための基準です。

接客の基本方針は次のとおりです。

- **気づき：** お客様の様子を見て、声をかけるか・そっと見守るかを判断する
- **簡潔さ：** 長い説明より、必要なことだけを伝える
- **尊重：** お客様のペースとプライバシーを最優先にする
- **自然さ：** 堅すぎず、でも丁寧に。Stayらしい落ち着いた距離感を保つ

---

## 施設の前提

**Stay はホテルとは別の施設です。**

| 施設 | マニュアル |
|------|-----------|
| **ホテル**（受付と客室が別場所） | 荷物運び・館内挨拶 → `ホテル_荷物運び.md` |
| **Stay**（本マニュアル） | 客室入室・館内ですれ違い など |

Stay の構成の目安：

- **1階：** 客室なし。ラウンジ・簡易キッチン・共用スペース
- **2階：** 客室（Room 01〜08）
- 入口は3箇所（焙煎入口・中央入口・スタジオ入口）

ホテル側の業務（荷物運び・フロント対応など）と混同しないよう、担当施設ごとにマニュアルを使い分けてください。

## 共通ルール（すべての場面に共通）

### 身だしなみ・態度

- 制服・エプロン・名札（着用時）を整え、髪・爪・口臭に気を配る
- お客様の方を向き、視線を合わせて話す（長時間の視線は避ける）
- 笑顔は「口角を少し上げる」程度でよい。無理な笑顔は不要
- 携帯電話の私用は、お客様の前では行わない

### 声かけ・挨拶

| 状況 | 基本の言い方 |
|------|-------------|
| 初めて目が合ったとき | 「こんにちは」「こんばんは」 |
| お客様から話しかけられたとき | まず「はい」と受け止め、内容に答える |
| お客様が会話を望んでいない様子のとき | 会釈のみ。追いかけて話しかけない |
| お客様が荷物・スマホ・会話中のとき | 話しかけず、会釈または待つ |

### 敬語の基準

- **お客様への呼び方：** 「お客様」「◯◯様」（予約名が分かる場合）
- **自分の言い方：** 「〜いたします」「〜させていただきます」
- **避ける表現：** 社内用語、他施設（ホテル）の名称、個人的な意見

### お客様のプライバシー

- 滞在目的・同伴者・会話内容を他のお客様やスタッフに口外しない
- 客室番号・部屋の中の様子を、必要以上に話題にしない
- SNSへの客室・お客様の無断撮影は禁止

## 1. お部屋に伺う際

客室清掃・備品補充・アメニティ配布・故障対応・忘れ物の届けなど、**2階客室**へ入るときのルールです。

### 入室前（必ず実施）

1. **ノック：** ドアを3回、ゆっくりと叩く
2. **声かけ：** 「失礼いたします。◯◯（担当部署名）の◯◯でございます」
3. **待機：** **最低10秒**待つ（返事がなくても、すぐに鍵を開けない）
4. **再確認：** 返事がなければ、もう一度ノックと声かけを行い、さらに10秒待つ
5. **入室判断：**
   - **清掃・点検（お客様不在想定）：** 社内ルールで入室可とされている場合のみ入室
   - **お客様在室が想定されるとき：** 返事があるまで入室しない。ドアの下に「伺いました」のメモを入れるなど、別手段を検討する

### お客様が在室のとき

- 入室後、まず会釈と挨拶
  > 「お邪魔いたします。◯◯のお手伝いに参りました」
- **目的を一言で伝える**（「タオルの補充です」「ご連絡いただいたエアコンの確認です」など）
- 作業場所・所要時間の目安を伝える
  > 「約◯分ほどお時間をいただきます」
- お客様の指示（「あちらに置いてください」「後でで大丈夫です」）を最優先する
- 作業中も必要以上に話しかけない。質問されたときだけ答える

### お客様が不在のとき

- 入室後も「失礼いたします」と一声（お客様が戻られた場合に備える）
- 私物・貴重品・スーツケースの中身には手を触れない
- ベッド・デスクの上の配置は、**動かす必要がある場合のみ**最小限に
- 退出時：エアコン・照明・窓の状態を確認し、ドアをしっかり閉める

### 清掃・メイク中にご在室のお客様がいる場合

- 「今から◯◯の清掃（交換）をさせていただいてもよろしいでしょうか」と必ず確認
- お客様が休憩中・仕事中と思われるときは、時間をうかがう
- バスルーム・寝室などプライベートな空間では、お客様がその場を離れる時間を確保するか、後回しにする

### お客様に会わずに退出するとき

- 伝言・メモが必要なときは、Stay指定の形式で残す（備品補充済み・点検完了など）
- 退出後、社内で「◯号室、在室のため未入室」など記録する（運用に合わせて）

## 2. 館内ですれ違った際

1階の共用スペース・階段・2階廊下など、**Stay 館内**でお客様とすれ違う・同じ空間にいるときのルールです。

### エリアごとの心がけ

| エリア | 心がけ |
|--------|--------|
| **1階**（ラウンジ・キッチン・ミーティングルームなど） | くつろいでいるお客様が多い。作業音・急ぎ足を控え、明るく挨拶する |
| **階段・通路** | 清掃道具を持ったまま急がない。お客様に道を譲る |
| **2階**（客室フロア・廊下） | 静かに。会話声・足音に配慮する。ノック以外の大声は避ける |

### 基本の動き

| 場面 | 対応 |
|------|------|
| 廊下ですれ違う | 道を少し譲り、会釈＋「こんにちは／こんばんは」 |
| お客様の後を歩く | 急がない。距離を取り、追い抜くときは「失礼いたします」 |
| お客様の前を歩く | 振り返りすぎない。曲がる角では先に案内の声をかけてもよい |
| 階段の上り下り | お客様に道を譲る。荷物を持っているお客様がいれば、先に通す |
| 清掃中 | 作業を一度止め、お客様が通り過ぎてから再開 |

### 共用スペース（1階ラウンジ・キッチンなど）

- お客様がくつろいでいるときは、**必要以上に近づかない**
- 清掃・補充は、お客様の邪魔にならないタイミングで行う
- お客様が調理・食事中のとき、匂いや音に配慮する
- 声かけの例（朝食・コーヒーなど）：
  > 「おはようございます。本日もよろしくお願いいたします」

### お客様から話しかけられたとき

1. 作業の手を止める
2. お客様の方を向く
3. 内容を聞き、分かる範囲で答える
4. 分からないことは「確認してまいります」と伝え、**必ず折り返すか担当者につなぐ**

よくある質問の目安：

| 内容 | 対応 |
|------|------|
| 施設の使い方（キッチン・シャワー・洗濯機） | `Stay.md` に沿って簡潔に説明。必要なら現地まで案内 |
| 周辺のお店・観光 | 施設で案内している範囲で。不確かな情報は避ける |
| 他のお客様について | 答えない。「申し訳ございません」と丁寧にお断り |
| クレーム・不具合 | まずお話を伺い、お詫びと対応方針を伝える。詳細は担当者へ引き継ぐ |
| ホテル側の問い合わせ | Stay とホテルは別施設。ホテルに関することはホテル担当者へ引き継ぐ |

### 避けるべき行動

- お客様の会話を聞き耳する・割り込む
- 廊下で大声で話す・私用の電話
- お客様の無断撮影
- 「いつチェックアウトですか？」など、プライバシーに踏み込む質問（業務上必要な場合を除く）

## 3. 困ったときのエスカレーション

次の場合は、**無理に自分だけで解決せず**担当者・責任者に連絡する。

- お客様が明らかに怒っている・強いクレーム
- 怪我・体調不良・緊急事態
- 客室内の備品破損・水漏れ・鍵のトラブル
- お客様の要望がマニュアルにない・判断に迷う
- お客様が入室を拒否したが、業務上どうしても対応が必要

連絡時に伝えること：**客室番号・お客様のお名前・状況・自分がすでにした対応**
