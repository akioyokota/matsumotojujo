## 1. 荷物運び（施設の前提と手順）

荷物運びの前提となる**施設の情報**（施設の前提）と、実際の**荷物運びの手順**です。

**施設の前提（重要）**

施設は、**チェックイン場所**と**ホテル**の2つに分かれます。

### チェックイン場所（ホテルとは別の場所）


| 名称           | 種別           | 役割                             |
| ------------ | ------------ | ------------------------------ |
| **おやきとコーヒー** | カフェ & レセプション | **チェックイン**を行う場所。お客様到着時に荷物を受け取る。チェックアウト時の荷物の保管・お渡し先でもある |


- お客様の宿泊場所（客室）ではない
- **チェックアウト手続き**はここではなく、客室の棟に応じたホテルのフロント（本箱／小柳）で行う

### ホテル（お客様が宿泊する建物）

荷物運びに関わる施設は次の通りです。


| 場所        | 役割                    |
| --------- | --------------------- |
| **本箱（西）** | 荷物の保管拠点及び滞在する客室のひとつ   |
| **小柳（東）** | 荷物の保管拠点及び滞在する客室のひとつ   |
| **湯小屋**   | 荷物の受取拠点のひとつ 宿泊者限定のお風呂 |


### 荷物について（荷物札・ミヨミヨ）

**荷物札について**

- **荷物札**は、お客様のお名前・部屋番号が書かれた荷物の識別する札です
- 荷物の受取時に、**お客様のお名前に合う荷物札**を探し、荷物**すべて**に付けます
- ホテル到着後、**荷物札の部屋番号の桁数**で棟を判断して届け先の客室を判断します
  - **部屋番号が2桁 → 本箱（西棟）**
  - **部屋番号が3桁 → 小柳（東棟）**
- 客室に荷物を届けたあと、ミヨミヨとあわせて外します

**ミヨミヨについて**

- **ミヨミヨ**は、荷物用のバンジーストラップ（ゴムベルト）です
- 受取時に、お客様の荷物**すべて**に付け、他のお客様の荷物と混ざらないようにします
- 客室に荷物を届けたあと外し、指定場所へ返却します

### 荷物運びの基本の流れ

お客様が**おやきとコーヒー**に到着して、荷物運びのまでの標準手順です。

**前提（重要）**

- お客様の荷物は、受取から客室へ入れるまで、**丁寧に扱う**

```
1. おやきとコーヒーで受取（荷物札・ミヨミヨを付ける）  →  2. 車でホテルへ運搬  →  3. 荷物札で棟を判断・客室前・入室  →  4. 定位置へ置く  →  5. 荷物札・ミヨミヨを外して完了処理
```


| ステップ               | いつ・どこで             | 何をするか                                                                                                                 |
| ------------------ | ------------------ | --------------------------------------------------------------------------------------------------------------------- |
| **1. 受取**          | おやきとコーヒー           | スタッフが荷物を受け取る。**お客様のお名前に合う荷物札**を探し、荷物**すべて**に**荷物札**と**ミヨミヨ**を付ける                                                      |
| **2. 運搬**          | おやきとコーヒー → ホテル     | フォローが必要な場合はインカムで部屋番号と「荷物をあげます」を伝えてから、お客様がチェックイン手続きをしている**あいだに**、**車**でホテルへ荷物を運ぶ（チェックインは**約15〜20分**。お客様が客室に着くまでに入れ終える） |
| **3. 棟の判断・客室前・入室** | ホテル → 該当する棟の客室前    | ホテル到着後、**部屋番号の桁数で棟を判断**（2桁＝本箱（西棟）／3桁＝小柳（東棟））し、該当する棟の客室前でノック・声かけのうえ、**数秒待ってから**入室する                                    |
| **4. 定位置へ置く**      | 本箱（西棟）または小柳（東棟）の客室 | 部屋番号を再確認し、荷物の種類に応じた**定位置**に置く                                                                                         |
| **5. 完了処理**        | ホテル内               | **荷物札とミヨミヨを外し**、ドアを閉め、インカムで完了報告する（下記参照）                                                                               |


#### 1. おやきとコーヒーで受け取る

- お客様がチェックインのために到着したら、**スタッフ**が荷物を受け取る
- **お客様のお名前に合う荷物札**を探し、お客様の荷物**すべて**に付ける
- あわせて、荷物**すべて**に**ミヨミヨ**（バンジーストラップ）を付ける
  - 他のお客様の荷物と混ざらないよう、受取の時点ですぐに付ける
- 荷物札の**部屋番号・お名前**が合っているか確認する（棟の判断はホテル到着後に行う）

お客様への声かけの例：

> 「お荷物はこちらでお預かりし、お部屋へお運びいたします」

#### 2. 車でホテルへ運搬する

- お客様がおやきとコーヒーで**チェックイン手続きをしている間**に運搬を行う
  - チェックインには**およそ15〜20分**かかる。この時間を目安に、お客様が客室に着くまでに荷物を入れ終えられるよう努める（状況によっては間に合わないこともあるが、できるだけ早く届ける）
- **車で運搬する前に**、ホテル側で荷物のフォローが必要な場合は、**インカムで「荷物をあげます」と部屋番号を伝えてから**出発する
  - 例：「000号室と0号室、荷物をあげます」
  - 複数件あるときは、対象の部屋番号をまとめて伝える
- **車**で、おやきとコーヒーからホテルへ荷物を運ぶ
  - **スーツケースは立てず、倒して積む**（立てたままだとカーブ時の遠心力で倒れるため）
- **車で運搬してきた荷物のフォローに行く人は、インカムで「行きます」と必ず応答する**（誰が対応するかを明確にし、対応漏れ・重複を防ぐ）
- ホテル到着後、**荷物札の部屋番号の桁数で棟を判断**し（**2桁＝本箱（西棟）／3桁＝小柳（東棟）**）、該当する棟の客室へ向かう
  - ホテル内の移動は、原則**従業員通路、EV**を使う
  - ただし、**荷物が多いとき**や**従業員用EVが混み合うとき**は、例外として**お客様用EV**を利用してよい
  - **客室鍵**は、次のいずれかでQRコードを確認し、自分のカメラで撮影、取り込みしたうえで、客室のQRコードで開ける
    - **LINE WORKS**の「**Matsumoto-jujo**」トークルームに、毎日添付されるQRコード
    - **中央東のホワイトボード**に掲示されている、QRが印刷されたもの
  - **靴はパントリーか客室の前で脱ぐ**

**所要時間の目安（積み込み〜客室へ入れるまで）**


| 出発地点             | 流れ                               | 目安時間 |
| ---------------- | -------------------------------- | ---- |
| **おやきとコーヒーから出発** | 積み込み → 運搬 → 客室へ入れる               | 約7分  |
| **ホテルから出発**      | おやきとコーヒーへ戻る → 積み込み → 運搬 → 客室へ入れる | 約10分 |


#### 3. 棟の判断・客室前・入室

まず**荷物札の部屋番号の桁数で棟を判断**し（**2桁＝本箱（西棟）／3桁＝小柳（東棟）**）、該当する棟の客室前へ向かう。客室前では次の手順で入室する。

1. **ドアを大きく3回ノック**
2. 声かけ（日本語）：
  > 「失礼いたします。お荷物をお持ちしました」
3. 声かけ（英語）：
  > 「Room service, I have your luggage.」
4. **数秒待ってから**ドアを開ける

**通常の荷物運びでは、返事を待つ必要はない。**  
お客様はおやきとコーヒーでチェックイン中のため、客室は**不在であることが多い**。ノックと声かけのあと、数秒待ってドアを開けて構わない。

次の場合のみ、**返事を待ってから**入室する。

- 荷物の届けに時間がかかり、すでにお客様が入室している可能性があるとき
- 声・気配などで、すでにお客様が客室にいるとわかるとき

#### 4. 定位置へ置く

**荷物札の部屋番号を再確認**してから、該当する棟（本箱（西棟）または小柳（東棟））の客室内の定位置に置く。


| 荷物の種類             | 置き場所         |
| ----------------- | ------------ |
| キャスター付き（スーツケースなど） | ドア付近の靴脱ぎ場あたり |
| 手鞄・バッグなど          | ソファ・荷物台へ     |


- 客室へ入れる際も、お客様の荷物は**丁寧に扱う**
- **定位置に置いたあと**、**客室内のメッセージカードのお客様のお名前**と、**荷物に付けた荷物札のお名前**が一致しているか必ず確認する
  - 名前が一致しない場合は荷物を回収し、部屋番号・荷物札・メッセージカードを見直す

#### 5. 完了処理

1. 付けた**荷物札**と**ミヨミヨ**を外す
2. **ドアをしっかり閉める**
  - **インスペクション前**に荷物を入れた場合も、客室を出るときは**必ず扉を閉める**（貴重品などの心配があるため、開けたままにしない）
3. **インカム**で、荷物を客室に入れたことを報告する
  - **荷物担当スタッフ**が報告する
  - 例：「（部屋番号）荷物入れました」
4. 外したミヨミヨはまとめて、**おやきとコーヒー または届け先の棟のフロント**（本箱（西）/小柳（東））へ返却する

#### 在室のお客様への挨拶（荷物届け時）

お客様が客室にいるときに荷物を届けた場合の挨拶例です。

**日本語**

> 「お待たせいたしました。お荷物は以上でお間違いございませんか？」
> 「ごゆっくりお過ごしくださいませ」

**英語**

> 「Sorry for waiting. Here's your luggage. Enjoy your stay!」

### ホテルに直接到着したお客様の荷物（アサイン表で確認）

おやきとコーヒーでのチェックインを経由せず、**お客様が直接ホテルに到着する**ケースもあります。この場合は荷物札がないため、**お名前とアサイン表**で届け先の部屋を確認します。客室への届け方（入室・定位置・完了処理）は「チェックイン時の基本の流れ」と同じです。

```
1. ホテルで受取・お名前を伺う  →  2. アサイン表で部屋を確認  →  3. 客室前・入室  →  4. 定位置へ置く  →  5. 完了処理
```


| ステップ            | いつ・どこで             | 何をするか                                                         |
| --------------- | ------------------ | ------------------------------------------------------------- |
| **1. 受取・お名前確認** | ホテル                | お客様の荷物を受け取り、**お名前を伺う**                                        |
| **2. 部屋の確認**    | ホテル                | 伺ったお名前を**アサイン表**と照らし合わせ、一致した**部屋番号**（＝棟）を確認する                 |
| **3. 客室前・入室**   | 該当する棟の客室前          | ノック・声かけのうえ、数秒待って入室する（手順は上記「3. 棟の判断・客室前・入室」と同じ。在室がわかるときは返事を待つ） |
| **4. 定位置へ置く**   | 本箱（西棟）または小柳（東棟）の客室 | 部屋番号を再確認し、荷物の種類に応じた**定位置**に置く                                 |
| **5. 完了処理**     | ホテル内               | ドアを閉め、**インカム**で完了報告する（例：「（部屋番号）荷物入れました」）                      |


- お名前が**アサイン表で確認できない**場合は、荷物を客室に入れず、フロント（本箱（西）/小柳（東））または責任者に確認する
- 荷物札がない場合でも、複数のお客様の荷物が混ざらないよう注意する（必要に応じて**ミヨミヨ**で区別する）
- **定位置に置いたあと**、**メッセージカードのお名前**と**アサイン表で確認したお客様のお名前**が一致しているか確認する

### 事前発送の荷物がある場合（宅急便）

お客様が**宅急便で事前に荷物をホテルへ発送している**ケースがあります。次の点に注意して対応します。客室への届け方（入室・定位置・完了処理）は「チェックイン時の基本の流れ」と同じです。

- **保管場所は金庫前**。届いた荷物は金庫前に保管する
- **基本チェックイン時間の15時までに**、荷物を部屋に入れる
- **宅急便伝票の依頼主（送り主）のお名前**と、**アサイン表のお名前**を照らし合わせ、一致した部屋（棟）を確認する
- **宅急便伝票はそのままの状態にする**（外したり剥がしたりしない）
- お名前が一致しない・確認できない場合は、部屋に入れず、フロント（本箱（西）/小柳（東））または責任者に確認する

### チェックアウト時の荷物運び

チェックアウト時に、お客様の荷物をホテルから**おやきとコーヒー（レセプション）**へ運ぶ手順です。お客様の荷物は、チェックイン時と同じく**丁寧に扱う**。

#### 基本の流れ

1. **チェックアウトは客室の棟のフロントで行う**
  - **本箱（西）**の客室 → **本箱（西）フロント**
  - **小柳（東）**の客室 → **小柳（東）フロント**
2. フロント担当が、お客様に**荷物をおやきとコーヒーへ運ぶかどうか**を伺う
3. 運ぶ希望があったお客様の荷物を、ホテルから**おやきとコーヒー**へ運ぶ
4. お客様の出発の仕方に応じて、下のどちらかでお渡しする

```
1. 各フロントでチェックアウト・運搬希望を確認  →  2. ホテルからおやきとコーヒーへ運搬  →  3. 保管してお渡し／または直接手渡し
```

| ステップ | いつ・どこで | 何をするか |
| --- | --- | --- |
| **1. チェックアウト・確認** | 本箱（西）または小柳（東）フロント | チェックアウト手続き。荷物をおやきとコーヒーへ運ぶかを伺う |
| **2. 運搬** | ホテル → おやきとコーヒー | 運ぶ希望のあった荷物を車などで運ぶ（スーツケースは倒して積む） |
| **3. お渡し** | おやきとコーヒー | 周辺散策がある場合は番号札で保管してお渡し。すぐ出発の場合は直接手渡し |

#### 周辺を散策してから出発するお客様（番号札で保管）

チェックアウト後、すぐに出発せず周辺を散策されるお客様も多い。この場合は次のとおり対応する。

1. 荷物に**番号札**を付ける
2. 対になっている**番号札**をお客様にお渡しする
3. 荷物は**おやきとコーヒー**で保管する
4. お客様が戻られたら、お持ちの番号札と荷物の番号札を照合し、一致した荷物をお渡しする

- 番号が一致しない・見つからない場合は、勝手に渡さずフロント・責任者へ確認する

#### すぐにおやきとコーヒーから出発するお客様（直接手渡し）

周辺を散策せず、おやきとコーヒーからすぐに出発するお客様には、おやきとコーヒーで荷物を保管せず、**待っているお客様へ直接手渡し**する。

## 2. お部屋に伺う際の挨拶

清掃・備品補充・故障対応・忘れ物の届けなど、**本箱（西棟）または小柳（東棟）の客室**へ入るときのルールです。荷物運び以外の入室時も、同じ基本を守ってください。

### 入室前（必ず実施）

客室前では、**在室・不在にかかわらず**、次の基本手順は必ず行います。

1. **ドアを大きく3回ノック**する
2. **用件に合わせた声かけ**をする：
  > 「失礼いたします。◯◯に伺いました」
  >
  > 例：
  >
  > - 清掃 → 「お部屋の清掃に伺いました」
  > - 備品補充 → 「アメニティーの補充に伺いました」
  > - 故障対応 → 「ご連絡いただいた件の確認に伺いました」
  > - 忘れ物の届け → 「お忘れ物のお届けに伺いました」
3. 状況に応じて、下の「在室がわかるとき／返事がないとき」に従って入室する

**英語（外国のお客様が多い客室の場合の参考）**

> 「Excuse me. [purpose].」
>
> 例：Housekeeping / Amenities / Maintenance / Delivery

#### 在室がわかるときの入室

声・気配・応答などで**お客様が部屋にいるとわかるとき**は、次を守ります。

- **スタッフ側から鍵で開けない**。お客様がドアを開けてくださるのを待つ
- 「どうぞ」「お入りください」など、**お客様の許可が出てから**入室する
- 許可がないままドアを開けたり、中に入ったりしない
- 「今はけっこうです」「後にしてください」と言われた場合は、無理に入らず、時間を改めて伺うか、フロント・責任者へ引き継ぐ

#### 返事がないときの入室

ノック・声かけを2回行っても返事がないときは、次のとおりです。

- お客様が不在の場合も、**フロント・責任者などから事前に入室許可が出ているときのみ**、鍵で開けて入室する
- 入室許可を確認できない場合は入室せず、フロント・責任者へ確認する
- 入室する直前にもう一度「失礼いたします」と声をかける
- **在室の可能性があるのに返事がない場合**（就寝中・入浴中など）は、鍵で無理に開けず、時間をずらすか、フロント・責任者に確認する
- 「起こさないでください」「後で」などの伝言がある場合は、入室せず対応を確認する

#### 入室時の共通ルール

- お客様がドアを開けてくださったときも、スタッフが鍵で開けるときも、**入室前に用件が伝わる声かけ**を行う
- ドアを開けたあと、すぐに室内へ踏み込まず、少し間を置いて様子を見る

### お客様が在室のとき

入室後、まず会釈と挨拶をする。


| 場面           | 例                                            |
| ------------ | -------------------------------------------- |
| 入室のあいさつ      | 「失礼いたします」                                    |
| 用件の説明        | 「◯◯に参りました」（例：タオルの補充、ご連絡いただいた不具合の確認、お忘れ物のお届け） |
| 作業前の確認       | 「今から◯◯させていただいてもよろしいでしょうか」                    |
| 退出（不具合対応の場合） | 「ご迷惑をおかけいたしました。」                             |
| 締めの挨拶        | 「ごゆっくりお過ごしくださいませ」                            |


### お客様が不在のとき

- **事前に入室許可が出ていることを確認してから**入室する
- 入室後も「失礼いたします」と一声（お客様が戻られた場合に備える）
- 私物・貴重品には手を触れない
- 退出時：ドアをしっかり閉める

## 3. 館内ですれ違った際の挨拶

**ホテル内**の施設、**おやきとコーヒー（ホテル外）**、およびそのあいだの移動路で、お客様とすれ違う・同じ空間にいるときの挨拶ルールです。

### エリアごとの心がけ


| エリア                  | 所在          | 心がけ                                                            |
| -------------------- | ----------- | -------------------------------------------------------------- |
| **おやきとコーヒー**         | ホテル外        | カフェとしてくつろいでいるお客様が多い。明るく挨拶し、作業音・急ぎ足を控える。チェックイン対応中のお客様には丁寧に声をかける |
| **本箱（西）フロント**        | ホテル内・本箱（西棟） | 明るく挨拶する。お客様の待ち時間に配慮し、急ぎ足・大声を避ける                                |
| **小柳（東）フロント**        | ホテル内・小柳（東棟） | 本箱（西）フロントと同様                                                   |
| **本箱（西棟）**（客室フロア・廊下） | ホテル内        | 静かに。会話声・足音に配慮する。作業中も一度止まる                                      |
| **小柳（東棟）**（客室フロア・廊下） | ホテル内        | 本箱（西棟）と同様                                                      |
| **湯小屋**              | ホテル内        | 通路・共用スペースとして、お客様に道を譲る                                          |
| **EV・パントリー**         | ホテル内        | ドアの開閉に注意。お客様を先に通す。「どうぞ」「失礼いたします」                               |
| **移動中**（棟間・エリア間）     | —           | 荷物・清掃道具を持ったまま急がない。お客様に道を譲る                                     |


### 時間帯による挨拶


| 時間帯         | 挨拶の例        |
| ----------- | ----------- |
| 朝（〜11時頃）    | 「おはようございます」 |
| 昼（11時〜17時頃） | 「こんにちは」     |
| 夕方以降        | 「こんばんは」     |


目が合ったら、会釈に加えて上記のいずれかを短く伝える。

### おやきとコーヒー（チェックイン場所）での挨拶

ホテルとは別の場所にあるカフェで、**チェックイン**を行います。カフェのお客様と宿泊のお客様が同じ空間にいるため、くつろいだ雰囲気を保ちながら丁寧に対応してください。


| 場面              | 挨拶の例                                                            |
| --------------- | --------------------------------------------------------------- |
| お客様が入店・受付へ向かう   | 「いらっしゃいませ」                                                      |
| チェックインの手続き開始    | 「ようこそお越しくださいました。◯◯様でいらっしゃいますね」                                  |
| ホテル客室への案内を伝えるとき | 「お部屋はホテルの◯◯棟にございます。こちらから◯◯へ向かっていただきます」（本箱（西棟）/小柳（東棟）と移動ルートを簡潔に） |
| 荷物をお預かりするとき     | 「お荷物はこちらでお預かりし、お部屋へお運びいたします」                                    |


チェックイン後は、お客様が**本箱（西棟）または小柳（東棟）の客室**へ向かう必要があるため、**どちらの棟か・客室までの行き方**を必ず伝えるか、ご案内する。

### 場面ごとの動きと挨拶


| 場面            | 動き           | 挨拶の例              |
| ------------- | ------------ | ----------------- |
| 廊下ですれ違う       | 道を少し譲り、立ち止まる | 会釈＋「こんにちは／こんばんは」  |
| お客様の後を歩く      | 距離を取り、急がない   | 追い抜くとき「失礼いたします」   |
| お客様の前を歩く      | 振り返りすぎない     | 曲がる角で「こちらへどうぞ」    |
| EVの乗り降り       | お客様を先に       | 「どうぞ」「お先に失礼いたします」 |
| 階段の上り下り       | お客様に道を譲る     | 会釈。荷物を持つお客様は先に通す  |
| 荷物運搬・清掃中      | 作業を一度止める     | 会釈。通り過ぎてから再開      |
| パントリー（靴の脱ぎ履き） | お客様がいれば譲る    | 「失礼いたします」         |


### お客様から話しかけられたとき

1. 作業の手を止める
2. お客様の方を向く
3. 内容を聞き、分かる範囲で答える
4. 分からないことは「確認してまいります」と伝え、適切な担当先へ引き継ぐ


| 内容              | 引き継ぎ先                                               |
| --------------- | --------------------------------------------------- |
| チェックイン          | おやきとコーヒー                                            |
| 施設・設備の使い方（ホテル内） | 客室の棟に応じ、本箱（西）フロントまたは小柳（東）フロントへ誘導。必要なら現地まで案内         |
| 周辺のお店・観光        | 施設で案内している範囲で。不確かな情報は避ける                             |
| 他のお客様について       | 答えない                                                |
| クレーム・不具合        | まずお話を伺い、お詫びする。詳細は客室の棟に応じたフロント（本箱（西）/小柳（東））・担当者へ引き継ぐ |


### 避けるべき行動

- 廊下・EV内で大声で話す・私用の電話
- お客様の会話に割り込む・聞き耳する
- お客様の無断撮影
- 返事を待つべき場面で、返事を待たずに客室のドアを開ける（荷物運びの通常時は除く。→ [3. 棟の判断・客室前・入室](#3-棟の判断客室前入室)）
- 「いつチェックアウトですか？」など、プライバシーに踏み込む質問（業務上必要な場合を除く）
