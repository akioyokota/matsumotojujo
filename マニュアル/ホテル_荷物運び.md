# ホテル お客様対応マニュアル

荷物運び・お部屋に伺う際・館内ですれ違った際の対応をまとめたマニュアルです。

---

## 施設の前提（重要）

施設は、**チェックイン場所**と**ホテル**の2つに分かれます。混同しないよう、次の定義を守ってください。

### チェックイン場所（ホテルとは別の場所）


| 名称           | 種別        | 役割（本マニュアルで関わる範囲）               |
| ------------ | --------- | ------------------------------ |
| **おやきとコーヒー** | カフェ（受付あり） | **チェックイン**を行う場所。お客様到着時に荷物を受け取る |


- お客様の宿泊場所（客室）ではない
- **チェックアウトはここでは行わない**（後述のホテル・フロントで実施）

### ホテル（お客様が宿泊する建物）

ホテルは**西棟**と**東棟**の2つの棟から構成されています。それぞれにフロントがあり、客室は棟ごとに分かれています。


| 棟      | フロント      | 客室    | 役割（本マニュアルで関わる範囲）            |
| ------ | --------- | ----- | --------------------------- |
| **西棟** | **西フロント** | 西棟の客室 | 荷物の受取の拠点のひとつ。荷物の届け先となる客室がある |
| **東棟** | **東フロント** | 東棟の客室 | 荷物の受取の拠点のひとつ。荷物の届け先となる客室がある |



| 名称      | 所在   | 役割（本マニュアルで関わる範囲）  |
| ------- | ---- | ----------------- |
| **湯小屋** | ホテル内 | 荷物の受取場所のひとつ（湯小屋前） |


※ チェックアウトは**西フロントまたは東フロント**（それぞれ西棟・東棟）で行います。本マニュアルの荷物運びとは直接関係しません。

### 荷物について（タグ・ミヨミヨ）

**タグについて**

- **タグ**は、お客様のお名前・部屋番号が書かれた荷物の識別札です
- 受取時に、**お客様のお名前に合うタグ**を探し、荷物**すべて**に付けます
- ホテル到着後、**タグの部屋番号の桁数**で棟を判断して届け先の客室を決めます
  - **部屋番号が2桁 → 西棟**
  - **部屋番号が3桁 → 東棟**
- 客室に荷物を届けたあと、ミヨミヨとあわせて外します

**ミヨミヨについて**

- **ミヨミヨ**は、荷物用のバンジーストラップ（ゴムベルト）です
- 受取時に、お客様の荷物**すべて**に付け、他のお客様の荷物と混ざらないようにします
- 客室に荷物を届けたあと外し、指定場所へ返却します

### エリアの関係（まとめ）

**チェックイン時の荷物運び**

```
【ホテル外】  おやきとコーヒー  …  お客様到着・荷物受取・チェックイン
                                （お名前に合うタグとミヨミヨを荷物すべてに付ける）
                    ↓（車で運搬）
【ホテル内】  ホテル到着  …  部屋番号の桁数で棟を判断（2桁=西棟／3桁=東棟）
                    ↓
              西棟 / 東棟の客室  …  荷物の届け先（届けたらタグとミヨミヨを外す）
                    ↓
              インカム  …  完了報告
```

※ 湯小屋前など、ホテル内での荷物受取がある場合は「[1. 荷物運び](#1-荷物運び)」のその他のパターンを参照してください。

エリアのあいだを移動するとき（棟をまたぐ移動・EV・廊下など）は、お客様より先に急がず、道を譲る・一声かけることを徹底してください。

【画像必要：おやきとコーヒー・ホテル（西棟/東棟・各フロント）の全体図・移動ルート】

## 共通ルール

### 身だしなみ・態度

- 髪・爪・口臭に気を配り、清潔感のある身だしなみを整える
- お客様の方を向き、視線を合わせて話す（長時間の視線は避ける）
- 携帯電話の私用は、お客様の前では行わない

### 敬語の基準

- **お客様への呼び方：** 「お客様」「◯◯様」（予約名が分かる場合）
- **自分の言い方：** 「〜いたします」「〜させていただきます」
- **避ける表現：** 社内用語、個人的な意見、他のお客様の情報

### 声をかけるかどうかの判断


| 状況                | 対応             |
| ----------------- | -------------- |
| 目が合った・すれ違った       | 会釈＋挨拶          |
| お客様が会話・電話中        | 会釈のみ。待つか、通り過ぎる |
| お客様が明らかに休憩・集中している | 必要な業務以外は声をかけない |
| お客様から話しかけられた      | 手を止め、向き合って対応する |

## 1. 荷物運び

### チェックイン時の基本の流れ

お客様が**おやきとコーヒー**に到着してチェックインする際の、荷物運びの標準手順です。

```
1. おやきとコーヒーで受取（タグ・ミヨミヨを付ける）  →  2. 車でホテルへ運搬  →  3. タグで棟を判断・客室前・入室  →  4. 定位置へ置く  →  5. タグ・ミヨミヨを外して完了処理
```


| ステップ          | いつ・どこで         | 何をするか                                                |
| ------------- | -------------- | ---------------------------------------------------- |
| **1. 受取**     | おやきとコーヒー       | スタッフが荷物を受け取る。**お客様のお名前に合うタグ**を探し、荷物**すべて**に**タグ**と**ミヨミヨ**を付ける |
| **2. 運搬**     | おやきとコーヒー → ホテル | お客様がチェックイン手続きをしている**あいだに**、**車**でホテルへ荷物を運ぶ（チェックインは**約15〜20分**。お客様が客室に着くまでに入れ終える）           |
| **3. 棟の判断・客室前・入室** | ホテル → 該当する棟の客室前      | ホテル到着後、**部屋番号の桁数で棟を判断**（2桁＝西棟／3桁＝東棟）し、該当する棟の客室前でノック・声かけのうえ、返事を待って入室する（下記参照） |
| **4. 定位置へ置く** | 西棟または東棟の客室     | 部屋番号を再確認し、荷物の種類に応じた**定位置**に置く                        |
| **5. 完了処理**   | ホテル内           | **タグとミヨミヨを外し**、ドアを閉め、インカムで完了報告する（下記参照）                        |


#### 1. おやきとコーヒーで受け取る

- お客様がチェックインのために到着したら、**スタッフ**が荷物を受け取る
- **お客様のお名前に合うタグ**を探し、お客様の荷物**すべて**に付ける
- あわせて、荷物**すべて**に**ミヨミヨ**（バンジーストラップ）を付ける
  - 他のお客様の荷物と混ざらないよう、受取の時点ですぐに付ける
- タグの**部屋番号・お名前**が合っているか確認する（棟の判断はホテル到着後に行う）
- 複数件あるときは、部屋ごとに仕分ける
【画像必要：ミヨミヨの付け方・タグとの位置関係】

お客様への声かけの例：

> 「お荷物はこちらでお預かりし、お部屋へお運びいたします」

#### 2. 車でホテルへ運搬する

- お客様がおやきとコーヒーで**チェックイン手続きをしている間**に運搬を行う
  - チェックインには**およそ15〜20分**かかる。この時間を目安に、お客様が客室に着くまでに荷物を入れ終えられるよう努める（状況によっては間に合わないこともあるが、できるだけ早く届ける）
- **車**で、おやきとコーヒーからホテルへ荷物を運ぶ
- ホテル到着後、**タグの部屋番号の桁数で棟を判断**し（**2桁＝西棟／3桁＝東棟**）、該当する棟の客室へ向かう
  - ホテル内の移動は、原則**従業員用EV**を使う
  - ただし、**荷物が多いとき**や**従業員用EVが混み合うとき**は、例外として**お客様用EV**を利用してよい
    - 繁忙時にお客様用EVを使う場合は**14:40まで**とする
  - 客室鍵が必要なときは**ホワイトボード**にあるものを使用する
  - **靴はパントリーで脱ぐ**
- 移動中に他のお客様とすれ違ったら、一度止まり道を譲る（→ [3. 館内ですれ違った際の挨拶](#3-館内ですれ違った際の挨拶)）

**所要時間の目安（積み込み〜客室へ入れるまで）**


| 出発地点             | 流れ                                 | 目安時間  |
| ---------------- | ---------------------------------- | ----- |
| **おやきとコーヒーから出発** | 積み込み → 運搬 → 客室へ入れる                  | 約7分   |
| **ホテルから始める場合**   | おやきとコーヒーへ戻る → 積み込み → 運搬 → 客室へ入れる | 約10分  |


#### 3. 棟の判断・客室前・入室

まず**タグの部屋番号の桁数で棟を判断**し（**2桁＝西棟／3桁＝東棟**）、該当する棟の客室前へ向かう。客室前では次の手順で入室する。

1. **ドアを大きく3回ノック**
2. 声かけ（日本語）：
  > 「失礼いたします。お荷物をお持ちしました」
3. 声かけ（英語）：
  > 「Room service, I have your luggage.」
4. **声がけ後、返事を待ってから**ドアを開ける

チェックイン手続き中のため、お客様がまだ客室にいないことが多い。不在の場合は社内ルールに従って入室する。

#### 4. 定位置へ置く

**タグの部屋番号を再確認**してから、該当する棟（西棟または東棟）の客室内の定位置に置く。

- 荷物を置く前に、**客室内のメッセージカードのお客様のお名前**と、**荷物に付けたタグのお名前**が一致しているか必ず確認する
  - 名前が一致しない場合は荷物を置かず、部屋番号・タグ・メッセージカードを見直す


| 荷物の種類             | 置き場所         |
| ----------------- | ------------ |
| キャスター付き（スーツケースなど） | ドア付近の靴脱ぎ場あたり |
| 手鞄・バッグなど          | ソファ・荷物台へ     |


- 客室の備品やお客様の私物は動かさない

#### 5. 完了処理

1. 付けた**タグ**と**ミヨミヨ**を外す
2. **ドアをしっかり閉める**
3. **インカム**で、荷物を客室に入れたことを報告する
  - **荷物担当スタッフ**が報告する
  - 例：「（部屋番号）荷物入れました」
4. 外したミヨミヨはまとめて、**おやきとコーヒー または届け先の棟のフロント（西/東）**へ返却する

### その他の受取パターン（ホテル内）

チェックイン以外で、ホテル内から荷物を受け取る場合は次のとおりです。客室への届け方（入室・定位置・完了処理）は上記と同じです。受取時にタグやミヨミヨが付いていなければ、**すべての荷物に付ける**。


| 受取場所      | 所在      | 備考                          |
| --------- | ------- | --------------------------- |
| **湯小屋前**  | ホテル内    | タグの部屋番号・お名前を確認し、該当する棟の客室へ運ぶ |
| **西フロント** | ホテル内・西棟 | フロントで荷物を受け取り、**西棟の客室**へ運ぶ   |
| **東フロント** | ホテル内・東棟 | フロントで荷物を受け取り、**東棟の客室**へ運ぶ   |


#### 在室のお客様への挨拶（荷物届け時）

**日本語**

> 「お待たせいたしました。お荷物は以上でお間違いございませんか？」
> 「ごゆっくりお過ごしくださいませ」

**英語**

> 「Sorry for waiting. Here's your luggage. Enjoy your stay!」

#### 【注意事項】

- [ ] **清掃が終わっていない部屋**は、荷物を入れたら鍵をしめ、清掃担当者へ引き継ぐ
- [ ] **冷蔵品**がある場合は、キッチンの冷蔵庫などを借りてわかりやすく引き継ぎする
- [ ] スーツケースを持ち上げる前に「失礼いたします」と一声
- [ ] 複数のお客様がいる場合は、代表の方の指示に従う

#### 【手順・その他メモ】


| ステップ           | メモ  |
| -------------- | --- |
| 1. おやきとコーヒーで受取（タグ・ミヨミヨを付ける） |     |
| 2. 車でホテルへ運搬    |     |
| 3. タグで棟を判断・客室前・入室      |     |
| 4. 定位置へ置く      |     |
| 5. タグ・ミヨミヨを外して完了処理        |     |

## 2. お部屋に伺う際の挨拶

清掃・備品補充・故障対応・忘れ物の届けなど、**西棟または東棟の客室**へ入るときの挨拶ルールです。荷物運び以外の入室時も、同じ基本を守ってください。

### 入室前（必ず実施）

1. **ドアを大きく3回ノック**
2. **声かけ：**
  > 「失礼いたします。◯◯（担当部署名）の◯◯でございます」
3. **最低10秒待つ**（返事がなくても、すぐに鍵を開けない）
4. 返事がなければ、もう一度ノックと声かけを行い、さらに10秒待つ
5. **返事を待ってから**ドアを開ける

**英語（外国のお客様が多い客室の場合の参考）**

> 「Excuse me. [Department name]. May I come in?」

### お客様が在室のとき

入室後、まず会釈と挨拶をする。


| 場面      | 日本語の例                                    |
| ------- | ---------------------------------------- |
| 入室のあいさつ | 「お邪魔いたします」                               |
| 用件の説明   | 「◯◯のお手伝いに参りました」（例：タオルの補充、ご連絡いただいた不具合の確認） |
| 作業前の確認  | 「今から◯◯させていただいてもよろしいでしょうか」                |
| 退出のあいさつ | 「お忙しいところ失礼いたしました。ほかにお手伝いできることはございますか」    |
| 締めの挨拶   | 「ごゆっくりお過ごしくださいませ」                        |


- お客様の指示（「あちらに置いてください」「後でで大丈夫です」）を最優先する
- 作業中は必要以上に話しかけない。質問されたときだけ答える
- 会話・電話中のときは、作業を控えるか、時間をうかがう

### お客様が不在のとき

- 入室後も「失礼いたします」と一声（お客様が戻られた場合に備える）
- 私物・貴重品には手を触れない
- 退出時：ドアをしっかり閉め、エアコン・照明の状態を確認する

### 客室内ですれ違う・顔を合わせたとき

客室の中でお客様と同じ空間にいるとき（清掃・補充・点検中など）は、次を心がける。


| 場面            | 対応                   |
| ------------- | -------------------- |
| 入室してお客様に気づかれた | 会釈＋「お邪魔いたします」        |
| 作業中にお客様が通る    | 作業を一度止め、道を譲る         |
| お客様と近い距離になる   | 「失礼いたします」と一声かけ、距離を取る |
| 退出するとき        | 「失礼いたしました」と会釈してから退出  |


### 声をかけない／控える場面

- お客様が就寝中・休憩中と思われるとき
- バスルーム・寝室などプライベートな空間（お客様がその場を離れるまで待つ、または後回しにする）
- 返事がないのに入室しようとするとき（必ず再ノック・再声かけ）

## 3. 館内ですれ違った際の挨拶

**おやきとコーヒー（ホテル外）**、**ホテル内**（西棟・東棟の各フロント・湯小屋・客室フロア）、およびそのあいだの移動路（棟をまたぐ通路・EV・廊下・パントリーなど）で、お客様とすれ違う・同じ空間にいるときの挨拶ルールです。

### エリアごとの心がけ


| エリア              | 所在      | 心がけ                                                            |
| ---------------- | ------- | -------------------------------------------------------------- |
| **おやきとコーヒー**     | ホテル外    | カフェとしてくつろいでいるお客様が多い。明るく挨拶し、作業音・急ぎ足を控える。チェックイン対応中のお客様には丁寧に声をかける |
| **西フロント**        | ホテル内・西棟 | 明るく挨拶する。お客様の待ち時間に配慮し、急ぎ足・大声を避ける                                |
| **東フロント**        | ホテル内・東棟 | 西フロントと同様                                                       |
| **西棟**（客室フロア・廊下） | ホテル内    | 静かに。会話声・足音に配慮する。作業中も一度止まる                                      |
| **東棟**（客室フロア・廊下） | ホテル内    | 西棟と同様                                                          |
| **湯小屋**          | ホテル内    | 通路・共用スペースとして、お客様に道を譲る                                          |
| **EV・パントリー**     | ホテル内    | ドアの開閉に注意。お客様を先に通す。「どうぞ」「失礼いたします」                               |
| **移動中**（棟間・エリア間） | —       | 荷物・清掃道具を持ったまま急がない。お客様に道を譲る                                     |


### 時間帯による挨拶


| 時間帯         | 挨拶の例        |
| ----------- | ----------- |
| 朝（〜11時頃）    | 「おはようございます」 |
| 昼（11時〜17時頃） | 「こんにちは」     |
| 夕方以降        | 「こんばんは」     |


目が合ったら、会釈に加えて上記のいずれかを短く伝える。

### おやきとコーヒー（チェックイン場所）での挨拶

ホテルとは別の場所にあるカフェで、**チェックイン**を行います。カフェのお客様と宿泊のお客様が同じ空間にいるため、くつろいだ雰囲気を保ちながら丁寧に対応してください。


| 場面              | 挨拶の例                                                    |
| --------------- | ------------------------------------------------------- |
| お客様が入店・受付へ向かう   | 「いらっしゃいませ」                                              |
| チェックインの手続き開始    | 「ようこそお越しくださいました。◯◯様でいらっしゃいますね」                          |
| ホテル客室への案内を伝えるとき | 「お部屋はホテルの◯◯棟にございます。こちらから◯◯へ向かっていただきます」（西棟/東棟と移動ルートを簡潔に） |
| 荷物をお預かりするとき     | 「お荷物はこちらでお預かりし、お部屋へお運びいたします」                            |


チェックイン後は、お客様が**西棟または東棟の客室**へ向かう必要があるため、**どちらの棟か・客室までの行き方**を必ず伝えるか、ご案内する。

### 場面ごとの動きと挨拶


| 場面            | 動き           | 挨拶の例              |
| ------------- | ------------ | ----------------- |
| 廊下ですれ違う       | 道を少し譲り、立ち止まる | 会釈＋「こんにちは／こんばんは」  |
| お客様の後を歩く      | 距離を取り、急がない   | 追い抜くとき「失礼いたします」   |
| お客様の前を歩く      | 振り返りすぎない     | 曲がる角で「こちらへどうぞ」    |
| EVの乗り降り       | お客様を先に       | 「どうぞ」「お先に失礼いたします」 |
| 階段の上り下り       | お客様に道を譲る     | 会釈。荷物を持つお客様は先に通す  |
| 荷物運搬・清掃中      | 作業を一度止める     | 会釈。通り過ぎてから再開      |
| パントリー（靴の脱ぎ履き） | お客様がいれば譲る    | 「失礼いたします」         |


### お客様から話しかけられたとき

1. 作業の手を止める
2. お客様の方を向く
3. 内容を聞き、分かる範囲で答える
4. 分からないことは「確認してまいります」と伝え、適切な担当先へ引き継ぐ


| 内容              | 引き継ぎ先                                       |
| --------------- | ------------------------------------------- |
| チェックイン          | おやきとコーヒー                                    |
| 施設・設備の使い方（ホテル内） | 客室の棟に応じ、西フロントまたは東フロントへ誘導。必要なら現地まで案内         |
| 周辺のお店・観光        | 施設で案内している範囲で。不確かな情報は避ける                     |
| 他のお客様について       | 答えない                                        |
| クレーム・不具合        | まずお話を伺い、お詫びする。詳細は客室の棟に応じたフロント（西/東）・担当者へ引き継ぐ |


### 避けるべき行動

- 廊下・EV内で大声で話す・私用の電話
- お客様の会話に割り込む・聞き耳する
- お客様の無断撮影
- 返事を待たずに客室のドアを開ける
- 「いつチェックアウトですか？」など、プライバシーに踏み込む質問（業務上必要な場合を除く）

## 4. 困ったときのエスカレーション

次の場合は、無理に自分だけで解決せず、**客室の棟に応じたフロント（西/東）または責任者**へ連絡する（チェックインに関することは おやきとコーヒー へ）。

- お客様が明らかに怒っている・強いクレーム
- 怪我・体調不良・緊急事態
- 客室内の備品破損・水漏れ・鍵のトラブル
- 判断に迷う・マニュアルにない要望
- お客様が入室を拒否したが、業務上どうしても対応が必要

連絡時に伝えること：**客室番号・お客様のお名前・状況・自分がすでにした対応**
